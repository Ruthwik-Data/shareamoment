## How to Use

### Option 1: View Locally (Immediate)
1. Download `ruthvik-portfolio.html` to your computer
2. Double-click the file to open it in your web browser
3. That's it! The portfolio is fully functional

### Option 2: Deploy to the Web (5 minutes)

**Recommended: Vercel (Free)**
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Click "Add New..." → "Project"
4. Drag and drop `ruthvik-portfolio.html`
5. Click "Deploy"
6. You'll get a live URL like `ruthvik-portfolio.vercel.app`

**Alternative: Netlify (Free)**
1. Go to [netlify.com](https://netlify.com)
2. Sign up
3. Drag and drop `ruthvik-portfolio.html` into the deploy zone
4. You'll get a live URL like `ruthvik-portfolio.netlify.app`

### Option 3: Get a Custom Domain (Optional)
1. Buy a domain like `ruthvikarepelly.com` from Namecheap ($12/year)
2. Connect it to Vercel/Netlify (they provide instructions)
3. Your portfolio will be live at your custom domain
