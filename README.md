A mobile-first AI chatbot inspired by Coca-Cola’s emotional branding strategy. The experience focuses on happiness, personalization, and real-time conversation through an interactive AI companion.

---

## ✨ Features

| Brand Strategy        | Implementation                              |
| --------------------- | ------------------------------------------- |
| Emotional Connection  | Mood-aware AI tone and empathetic responses |
| Happiness & Lifestyle | Occasion-based drink recommendations        |
| Personalization       | “Share a Coke” name capture                 |
| Brand Identity        | Coca-Cola inspired red-white UI             |
| Interactive UX        | Quick reply cards and smooth animations     |
| Mobile First Design   | Fully responsive chat experience            |

---

## 🎨 Core UX Highlights

* Mood-based dynamic UI colors
* Contour bottle inspired chat bubbles
* Animated typing indicator
* Dark / Light mode support
* Quick reply interaction cards
* Personalized onboarding experience
* Smooth UI animations
